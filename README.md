# 认养农民人格测评网页

这是一个可直接打开的静态网页原型，用于模拟“禾作计划”认养消费人格测试。

## 文件

- `index.html`：页面结构
- `styles.css`：视觉样式
- `app.js`：题目、人格计算、商品推荐逻辑
- `assets/farm-adoption-hero-cartoon.png`：新版卡通首屏视觉图
- `assets/farm-adoption-hero.png`：旧版写实视觉图，保留用于对比

## 可替换内容

- 在 `app.js` 的 `questions` 中修改题目和选项权重。
- `questions` 支持普通文字题，也支持 `type: "visual"` 的图片直觉题和 `type: "sound"` 的听感选择题。
- 在 `typeProfiles` 中修改人格名称、人格描述和策略建议。
- 在 `typeProfiles.avatar` 中修改每种人格的人物卡通形象元素。
- `products` 已按禾作计划/同类认养平台常见销售品类建立样本库，包括水果、蔬菜盒子、玉米、艾草、鸡蛋、粮食、咖啡园和羔羊认养。
- `products.link` 可填小红书店铺链接、微信小程序口令或具体商品链接；网页端普通浏览器不一定能直接打开小程序口令。

推荐先把真实商品按这四个维度标注：

- `risk`：固定交付为负，产量波动为正
- `engage`：省心托管为负，高参与/采摘/直播为正
- `proof`：弱溯源为负，强证据链/检测/售后为正
- `value`：性价比导向为负，助农/生态/教育导向为正
